# AI

1. AI是什么？

    - AGI，通用人工智能（Artificial General Intelligence）
    - AIGC，AI生成内容（AI Generated Content）

    - NLP，自然语言处理（Natural Language Processing）
    - LLM，即大型语言模型（Large Language Model）
    - ChatGPT，（Chat Generative Pre-trained Transformer）
